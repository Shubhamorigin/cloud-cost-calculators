# Hidden Cloud Costs & Optimization Calculators (2026)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

Cloud providers often hide their most expensive pricing tiers, bandwidth taxes, and overage fees in the fine print. This open-source repository exposes the real compute, pipeline, and bandwidth costs of popular Developer Tools and Cloud platforms.

Below is a curated collection of 12 interactive calculators and data sets to help you optimize your infrastructure and stop overpaying.

---

## Contributing
This repository serves as an open directory. Did a cloud provider change their pricing? Did you build a better cost calculator? Let's keep this data accurate for the community!

To contribute an update or add a new tool:
1. Fork this repository
2. Edit the `README.md` file with the updated pricing data or tool link
3. Open a Pull Request with a brief explanation of the change
